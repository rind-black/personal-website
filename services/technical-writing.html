Concepts Guide · Tarantool
Platform › Concepts
Storage Engines
A storage engine is a set of low-level routines that store and retrieve values. Tarantool offers a choice of two storage engines:
· memtx — the in-memory storage engine used by default.
· vinyl — the on-disk storage engine.
Data Model
Tarantool is a NoSQL database. It stores data in spaces, which can be thought of as tables, and tuples, analogous to rows.
Note: For details on persistence and recovery, see the Data Persistence section.

Installation Guide
Installation Guide: DNS Configuration
Before installing Mailion, configure the following DNS records for your domain. All records must propagate before proceeding.
# Required DNS records
MX   @   mail.yourdomain.com   Priority 10
TXT   @   "v=spf1 include:_spf.yourdomain.com ~all"
TXT   _dmarc   "v=DMARC1; p=quarantine"
1
Log into your DNS provider and navigate to DNS Management.
2
Add the MX record pointing to your mail server hostname.
Note: DNS propagation can take up to 48 hours. Verify with nslookup.
