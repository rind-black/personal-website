Installation Guide: DNS Configuration
Before installing Mailion, configure the following DNS records for your domain. All records must propagate before proceeding to the next step.
Required DNS Records
# Type    Host                   Value
MX      @     mail.yourdomain.com   Priority 10
TXT     @     "v=spf1 include:_spf.yourdomain.com ~all"
TXT     _dmarc   "v=DMARC1; p=quarantine; rua=mailto:dmarc@yourdomain.com"
CNAME   mail    yourdomain.com
1
Log into your DNS provider and navigate to DNS Management.
2
Add each record from the table above. Leave TTL at the provider default unless instructed otherwise.
3
Verify propagation: nslookup -type=MX yourdomain.com
⚠ Note: DNS propagation can take up to 48 hours. Do not proceed with Mailion installation until all records resolve correctly.
